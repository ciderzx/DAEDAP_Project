# DAEDAP_Project
<h1 align="left">Unity Project [ DAEDAP ]</h1>

<h3> Game Information </h3>

+ 이름 : DAEDAP
+ 플랫폼 : Android 
+ 조작 방식 : Touch & Joystick Touch
+ 이야기 내용 : 용사가 적들을 물리치며 악마 or 천사를 잡는 게임
+ 진행 방식 : 맵이 랜덤으로 생성되고 3스테이지까지 클리어 하면 보스 스테이지로 구성
